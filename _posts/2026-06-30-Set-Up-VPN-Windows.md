This guide explains how to set up a working VPN for Windows. 
打开glados网站，长这样：：（我打开的是cloud那个）
![Picture](/random-blog-test/assets/vpn/vpn-1.png)

往下翻，你就会发现一些链接：
选择标注跟clash相关的，多复制几个
链接的作用：这些链接里面藏着翻墙线路，时不时就更新，不是固定的，所以我们要传入链接，clash读入链接自己就随时更新节点信息了，不需要我们手动更新ip，这就是为什么glados给我们链接但不给具体ip。

![Picture](/random-blog-test/assets/vpn/vpn-2.png)

接下来我们下载clash verge：（这个文档在qq邮箱，点击一下就自动下载了）

![Picture](/random-blog-test/assets/vpn/vpn-3.png)

下载完了以后我们就打开clash，点击订阅，然后就可以把刚才复制好的glados链接传入了

![Picture](/random-blog-test/assets/vpn/vpn-4.png)

然后改一下当前节点，随便改就行

![Picture](/random-blog-test/assets/vpn/vpn-5.png)

当前节点点开长这样：：随便点一个

![Picture](/random-blog-test/assets/vpn/vpn-6.png)
